<h1>@gamjajeon/console-utils</h1>
<span>oz 교육2팀 자바스크립트 교육 컨텐츠 전용 패키지입니다. 콘솔 입출력을 간단하게 사용할 수 있도록 도와주는 유틸리티 함수들을 제공합니다.</span>
